El proyecto es en .net core 6.0.

Se corre en visual code.

Tienes que ir a la carpeta raiz del proyecto y realizar el comando "dotnet run" para poder correrlo.

En la carpeta ".\Inputs". Esta los archivos de texto de las entradas y salidas del programa.
En el archivo ".\Inputs\Inputs.txt" estan las entradas.
En el archivo ".\Inputs\Outputs.txt" estan las salidas.
